# Web Portal (форум/соцсеть + инструменты)

## 1) Роли веба
1) Сообщество:
- форум (обсуждения, вопросы/ответы, опыт)
- профили пользователей/участков (опционально)
2) Инструменты:
- план участка на большом экране
- импорт/экспорт, массовые правки
- сезонные отчёты (погода, урожай, задачи)
- **диаграмма сезона (посадки → рост → сбор)**
- экспериментальные AI‑инструменты (разметка по фото)

## 2) Рекомендуемая реализация
- Cloud API — единый бэкенд
- Web SPA (Next.js/React) — кабинет и инструменты
- Forum engine (Discourse или аналог) — отдельно, интеграция по SSO (OIDC)

## 3) MVP веба
- лендинг + документация
- вход через OIDC
- просмотр плана участка и календаря (read‑only)
- форум как отдельный продукт (минимальная интеграция аккаунта)
