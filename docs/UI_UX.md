# UI/UX: экраны и адаптивность (v0.5)

Документ описывает минимальные экраны ядра и ключевые сценарии. UI проектируется **адаптивным**: телефон → планшет → web.

## 1) Навигация (Android)
Телефон:
- Bottom Navigation: Главная / План / Календарь / Посадки / Журнал
- Остальное: через меню (Зоны, Настройки, Справочники)

Планшет:
- Navigation Rail / Drawer
- Multi‑pane, где возможно: слева список, справа карточка.

## 2) Ядро (обязательные экраны)

### 2.1 Главная (Today / This week)
- ближайшие задачи (с приоритетами)
- предупреждения (заморозок/ливни/дефицит влаги)
- быстрые действия: добавить посадку, запись, фото‑наблюдение, задачу

### 2.2 План участка (Plot Planner v1)
Назначение: визуально видеть “что где растёт”.

MVP:
- список зон + режим “схема”:
  - пользователь создаёт “контуры” зон (простые прямоугольники/полигоны) или выбирает шаблон (теплица, грядки)
  - размещает посадки (точками/группами) внутри зон
- подсказка масштаба (опционально): “1 клетка = 0.5 м”

Планшет:
- drag&drop посадок на схему
- зум/панорамирование

### 2.3 Посадки
- список + фильтры (зона, культура, статус, “требует внимания”)
- карточка посадки:
  - параметры (культура/сорт/даты)
  - прогноз/окно урожая (ориентировочно)
  - связанные задачи
  - журнал/фото

### 2.4 Календарь и задачи
- список задач на неделю + календарь месяц
- отметка “сделано”
- создание пользовательских задач
- “предложенные задачи” (из правил/ИИ) — отдельным блоком с подтверждением



#### 2.4.1 Диаграмма сезона (Timeline/Gantt)
- альтернативный режим календаря: “Календарь / Диаграмма”
- строки: посадки (культура/сорт/зона)
- сегменты:
  - период роста (посадка → начало окна сбора)
  - окно сбора (диапазон)
  - ключевые точки/задачи (всходы, пересадка, обработки и т.п.)
- горизонтальная прокрутка по времени + вертикальная по списку
- линия “сегодня”
- фильтры: участок/зона/культура/статус
- тап по сегменту → карточка посадки / задачи

См. docs/TIMELINE_DIAGRAM.md
### 2.5 Журнал
- лента записей
- добавление записи (текст + фото + теги)
- привязка к посадке/зоне

### 2.6 Участок: климат/погода (ядро)
Экран “Климат участка”:
- регион/координаты
- оценка: последний/первый заморозок, длина сезона (если посчитано)
- прогноз на 7–10 дней: минимумы/осадки
- настройки уведомлений: “заморозки”, “осадки/полив”

## 3) Расширения (экраны по мере появления)

### 3.1 AI‑помощник (чат)
- вход: вопрос + фото (опционально) + выбор посадки/зоны
- выход: ответ + шаги + “предложения задач” (кнопки “добавить в календарь”)

### 3.2 Обрезка
- мастер: культура → сезон → цель → фото(2–3)
- результат: инструкция + (позже) подсветка зон на фото

### 3.3 Болезни/вредители
- мастер: культура/условия → фото → уточнения
- результат: гипотезы + проверки + безопасный план + предложения задач

### 3.4 Справочники
- культуры/сорта
- техкарты выращивания
- СЗР/удобрения (поиск/фильтры; без “самодеятельных” рецептов)
- почвы/плодородие (план работ)

### 3.5 Заготовки/рецепты
- предложения при сборе урожая
- подбор рецептов по культуре/объёму

### 3.6 Автоматизация
- подключение (Home Assistant / MQTT)
- “привязать задачу к действию” (полив)

## 4) Web Portal (UI)
- форум/вопросы‑ответы (встроенный движок или Discourse)
- большой план участка (drag&drop, печать, экспорт)
- массовые правки и импорт
- аналитика сезона (графики, отчёты)
