# План участка с посадками (Plot Planner)

## 1) Цель
Дать пользователю визуальную модель участка:
- где какие зоны (грядки/теплица/сад/клумбы);
- где размещены посадки;
- быстро видеть загруженность зон и севооборот.

## 2) MVP (v1)
Функции:
- создать план участка (PlotPlan) для Site;
- добавить/редактировать контуры зон (полигоны/прямоугольники) и связать их с Zone;
- разместить посадки (Planting) на плане:
  - точка (одиночное растение/куст/дерево),
  - группа (ряд/грядка) как прямоугольник/линия с количеством и шагом.

Ограничения MVP:
- координатная система локальная (0..1) или “метры” по шкале, которую задаёт пользователь;
- без “точной геодезии”: это схема, а не кадастр.

## 3) Продвинуто (v2+)
- импорт подложки (фото/скан плана) как фон;
- инструменты выравнивания/масштаба/север;
- “слои”: постройки, деревья, дорожки, водоём, тени;
- совместимость культур на плане (подсветка конфликтов/семейств);
- генерация “авто‑размещения” по правилам (экспериментально);
- web‑редактор плана (drag&drop на большом экране).

## 4) Данные
Сущности:
- PlotPlan: план (1..n на участок)
- PlotFeature: геометрия (GeoJSON) для зон и объектов
- PlantPlacement: размещение посадок на плане

## 5) Интеграция с календарём
- при выборе зоны на плане → показать задачи и посадки;
- при добавлении посадки → предложить сразу “положить на план”.
