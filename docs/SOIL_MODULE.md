# Почва: карта, лаборатория, плодородие

## 1) Уровни данных
1) MANUAL: пользователь ввёл тип/дренаж/pH.
2) MAP: почвенная карта даёт фоновый тип (coarse).
3) LAB: анализ почвы (с датой, глубиной, методикой, единицами).

## 2) Почвенная карта
- источник выбирается через провайдер (интерфейс)
- карта используется как предположение, всегда доступно ручное исправление

## 3) Лабораторный анализ
Требования:
- хранить единицы измерения и методику (если указана)
- не пытаться “пересчитывать” в единый стандарт без профиля метода

Показатели (минимум):
- pH (указать тип: водный/солевой)
- органическое вещество/гумус
- P и K (доступные формы)
Опционально:
- Ca, Mg, микроэлементы, EC

## 4) Восстановление плодородия (MVP)
Цель: безопасный и понятный план работ без “агрохимии в миллиграммах”.

Состав протокола:
- компост/органика/мульча
- сидераты и покровные культуры
- севооборот (по семействам)
- структура и уплотнение (меры без риска)

Результат:
- план → задачи в календаре → отметки выполнения → повтор

Продвинуто:
- интерпретация анализов по методике
- расчёт коррекций (известкование и т.п.) только при валидной модели и дисклеймерах
